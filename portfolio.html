My past portfolio/project

Study Buddy

Study Buddy is a web application developed through vibe coding. This project is developed during a workshop called "VibeCoding with AI" organised by CGMA. In this workshop, I have been exposed to many AI assisted application that are useful in developing applications such as Google Stitch for UI/UX generation, Antigravity, a AI-powered IDE, Supabase for backend management and Netlify for deployment.

  In this application, user can produce flashcards that helpful for quick revisions of topic. The flashcards can be produced either by uploading the notes or typing the topic covered. Other than tha, user can also challenge themselves with auto-generated quiz.

AR & 3D model Projection

This AR website is created during a workshop that exposed us to Augmented Reaity (AR) and 3D model projection using AR. In this workshop, we were expsoed with few 3D model creation and rigging application. Then, we were exposed with how to produced a website that can project 3D model using AR specifically AR.js.

  We were exposed on how to create a custom marker for the marker for AR projection. We were also exposed on how to set up the 3D model to be projected such as the size, the orientation, the color, or even using imported model. Lastly, we were also exposed on how to create interaction between the projected 3D model such as changing color when two model were brought closely to each other.

Kingdom Blitz

Kingdom Blitz is a game project for my degree course "Programming Technique II". In this project, we are required to use object-oriented programming, which involve classes and also window BGI library for the graphical drawing of the game.

  In this game, the player is a knight with a gun, where user can aim and shoot using thier mouse position. The enemies will continuously spawn and get close to player to damage the player. The game have few different type of enemies that spawn at certain time interval. At the 10:00 minute time interval, a boss will spawn, where defeating the boss will causes the player to clear the game.
